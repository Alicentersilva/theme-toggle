# Toggle

Projeto desenvolvido com HTML, CSS e JavaScript para praticar a criação de componentes interativos de interface.

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## Funcionalidades

* Alternância entre tema claro e escuro
* Atualização dinâmica da interface
* Interação com eventos JavaScript

## Objetivo

Praticar manipulação do DOM, eventos e alterações visuais em páginas web utilizando JavaScript.

## Observação

Projeto desenvolvido para fins de estudo e prática com base em um tutorial educacional.
